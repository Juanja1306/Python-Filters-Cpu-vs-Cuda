# python-filters-cpu-vs-cuda
Filtros de imagen en Python con dos rutas: baseline secuencial en NumPy y aceleración CUDA con PyCUDA. Incluye Gaussian, box, mediana, Sobel/Scharr, Laplaciano y convoluciones N×N, más un banco de pruebas reproducible (CPU vs. GPU) con métricas y gráficos. Perfecto para aprender y comparar rendimientos.
